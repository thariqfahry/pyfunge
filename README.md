# Pyfunge
Yet another Befunge-93 interpreter written in Python.

## Running 

```sh
python main.py /path/to/file.bf
```

## Example: Hello, World!
`./examples/helloworld.bf`
```befunge
"!dlroW ,olleH">:v
               |,<
               @
```

```
Hello, World!
```
